# NW-Shuriken

A collection of components for building the http://ninjawars.net webgame.
[Build no longer available on codeship]

## Install

    yarn install

## Build

    yarn build

## Test

    yarn test

## Lint

    yarn lint

## Deploy

    yarn deploy 

Deploys to http://nw-shuriken.surge.sh

No production deployment yet.

## Publish

    yarn login
    yarn publish
